HTML: Langages de marquage

Definition:

• HTML est un langage de marquage (Markup Language) permettant la description de données hypertextes.

• Ces documents HTML présentent une fonctionnalité supplémentaire par rapport aux autres documents.

• Un langage de marquage est composé des :

• Balises (éléments) • Attributs • Entités
  1. Balise:
    2.         ◗ Les balises sont les blocs de données principaux des documents HTML
      
            ◗	Les balises peuvent contenir du texte, d'autres balises, ou être vides      
            ◗	Utilisent les délimiteurs ‘ <‘ et ‘ >‘

    HTML Example:

    <!--Commentaires en HTML -->
    <b> Bonjour </b> 
            ◗ Les balises en paires: Elles s'ouvrent, contiennent du texte, et se ferment plus loin.  
              • Exemple <p> et  <div> .  
            ◗ 	Les balises orphelines: Ce sont des balises qui servent le plus souvent à insérer un élément à un endroit précis.  
              • Exemple <br> et  <img> .

    HTML Example:

    <div> <p> Ceci est une paragraphe </p> </div>
    <br >
    <img src = "../../images/users.png" >
  2. Les attribut:
    3.         ◗	Sont un peu les options des balises. 
    ◗  Complètent les balises pour donner des informations supplémentaires  
            ◗ 	Les balises orphelines: Ce sont des balises qui servent le plus souvent à insérer un élément à un endroit précis.  
     ◗ Se placent après le nom de la balise ouvrante et ont le plus souvent une valeur 
           • Un attribut d’une balise a :
    • Un nom
    • Une valeur
    <font color = "#FF0000" >
    <img src = "logo.png" >
